At a glance

  • Cloud — Open · sign in · data anywhere

Two doors

You can either explore with sample data (nothing saved) or create an account and own your data in the cloud.

DoorYou getDo
TrySample · nothing savedOpen → Try app → explore
Sign upYour data · cloudOpen → Create account → CRM

Try = peek. Sign up = own.

Try — 3 steps

  1. Open app (any device)
  2. Click Try the app
  3. Explore · sample data · no account

<2 min. Ready? → Sign up.

Sign up — 5 steps

Create an account, then Organizations (create/join/open org), set company name and brand tone in Onboarding, then land on the Dashboard. Everything runs in the cloud; no installs.

  1. Open → Get started or Sign In
  2. Create account · optional 2FA
  3. Connection → Skip → Continue
  4. Onboarding: Company + tone → Save
  5. Dashboard · your cloud

Build data (order)

Suggested order: add Companies first (accounts you sell to), then Leads, then Pipeline deals; convert leads to contacts/deals, then add Tasks and log Activities.

#DoExample
1CompaniesAcme, TechStart
2Leads · company · source · statusAlex, website, New
3Pipeline · name · value · stageAcme deal, $50k
4Contacts · or Convert lead → contact + dealConvert Alex
5–6Tasks · ActivitiesFollow up · Intro call

Core workflow

  1. Dashboard → type · goal · context → Generate
  2. Generated → edit · Send to CRM
  3. Send to CRM → Contact/Deal → Confirm
  4. History → Copy · Regenerate · Send again

Templates = pre-set type + goal.
