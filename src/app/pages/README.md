# Pages

Route-level components for the ACI frontend. Each page has a short README below and is summarized in [../reports/FRONTEND_PAGES_REPORT.md](../reports/FRONTEND_PAGES_REPORT.md).

| Route | Page | README | Purpose |
|-------|------|--------|---------|
| `/` | Homepage | [Homepage.README.md](./Homepage.README.md) | Landing: hero, features, CTA to sign in |
| `/login` | Login | [Login.README.md](./Login.README.md) | Sign in (demo / Google) → dashboard |
| `/connect` | Connection | [Connection.README.md](./Connection.README.md) | Connect CRM account |
| `/onboarding` | Onboarding | [Onboarding.README.md](./Onboarding.README.md) | Set company name & brand tone |
| `/dashboard` | Dashboard | [Dashboard.README.md](./Dashboard.README.md) | Generate copy: type, goal, context → AI |
| `/generated` | GeneratedCopy | [GeneratedCopy.README.md](./GeneratedCopy.README.md) | View/copy/regenerate/send generated text |
| `/send` | SendToCrm | [SendToCrm.README.md](./SendToCrm.README.md) | Attach copy to contact or deal in CRM |
| `/templates` | Templates | [Templates.README.md](./Templates.README.md) | Browse templates → use on dashboard |
| `/history` | History | [History.README.md](./History.README.md) | List and copy from copy history |
| `/settings` | Settings | [Settings.README.md](./Settings.README.md) | User settings, connection, delete account |
| `/help` | Help | [Help.README.md](./Help.README.md) | How it works, FAQ |
| `/privacy` | Privacy | [Privacy.README.md](./Privacy.README.md) | Privacy policy |
| `/terms` | Terms | [Terms.README.md](./Terms.README.md) | Terms of service |

All pages use `MAIN_CONTENT_ID` from `SkipLink` for accessibility (skip to main content).
